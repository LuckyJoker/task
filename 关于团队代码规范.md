# 关于团队代码规范

### 工具的安装和使用

1. **eslint、tslint和prettier**

   全局安装

   `npm i -g eslint babel-eslint eslint-plugin-import eslint-plugin-react eslint-plugin-jsx-a11y eslint-config-airbnb tslint prettier`

   配置文件
   - eslint配置一个独立的`.eslintrc.yaml`文件，放在用户文件夹的根路径下

   ```yaml
   ---
   extends: airbnb
   parser: babel-eslint
   plugins:
    - react
    - jsx-a11y
    - import
   env:
    amd: true
    browser: true
    jquery: true
    node: true
    es6: true
    worker: true
   rules:
    no-console: off
    max-len: off
    require-jsdoc: warn
    func-names: off
   ```

